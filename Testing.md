# Unit Testing

The `testing` folder contains unit tests using [GoogleTest](https://google.github.io/googletest/primer.html).

## Makefile 
```
$ cd testing
```

- Create executables for each test 
    ```
    $ make all
    ```
- Create single executables for class foo
    ```
    $ make foo_test
    ```
- Clean
    ```
    $ make clean
    ```
